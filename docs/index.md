# isroot.in
::: warning
This docs are still in **beta** .feel free to report issues
:::
isroot.in is a free subdomain platform for developers to deploy projects, APIs, and demos instantly—no credit card required.

## Quick start

#### Getting your subdomain : 
- **Sign in** with your github account at [https://isroot.in/signin](https://isroot.in/signup)
- Then register your subdomain ``eg:yourname.isroot.in``
- Your domain is ready to use !
